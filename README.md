# CTF platform
## Vue.js (Frontend), Node.js (Backend), PostgreSQL

- Frontend
    - Vue.js, Vue Router, BootstrapVue

- Backend
    - Node.js, Express, TypeORM, Socket.io

## Setup for development
Client
```
cd client
npm install
```

Server
```
cd server
npm install
```

Shared
```
cd shared
npm install
```

Environment variables can be changed in .env files


## How to run services
Client, Server and shared
```
npm run dev
```
